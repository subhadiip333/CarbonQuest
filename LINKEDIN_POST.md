# LinkedIn Post Draft

Excited to announce **CarbonQuest AI**, an enterprise-grade AI Sustainability Coach I engineered! 🌍🚀

Tackling carbon emissions is a daunting challenge. We often don't know the exact impact of our daily choices, and traditional static carbon calculators fail to drive long-term habit formation. That's why I completely reimagined the experience by building **CarbonQuest AI**—a context-aware intelligent companion that blends fitness-tracker gamification with advanced AI orchestration.

Built with a scalable Next.js and Node.js microservices architecture, the platform solves sustainability tracking through several deeply technical features:

✨ **Gemini 2.5 Flash NLP Engine**: Instead of tedious manual data entry, users log activities via free-text natural language (e.g., "Commuted 15km to the office by electric scooter"). The backend Gemini proxy parses the context, extracts the exact action categories, and dynamically calculates CO₂ cost and savings using complex prompting logic.

✨ **Carbon Twin Digital Ecosystem**: A state-machine-driven digital avatar that visually heals or degrades (from "Polluted" to "Pristine") based on the user's real-time computed sustainability score.

✨ **Smart Multi-Modal Routing**: Integrated with the Google Maps Directions and Places Autocomplete APIs. It calculates real-time distances and contrasts the precise carbon footprint of driving a gasoline car vs. taking public transit vs. bicycling.

✨ **AI Predictive Forecasting**: An analytical engine that aggregates your historical carbon data to predict your 30-day emissions trajectory, providing actionable, intelligent insights on how to reverse negative trends.

✨ **Context-Aware Quests & Eco Marketplace**: The system dynamically generates personalized daily challenges (powered by Gemini) based on weather and lifestyle constraints. Users earn "Green Points" and "Coins" to climb the global Leaderboard and redeem virtual currency for real-world impact actions like tree planting.

To ensure production readiness, the entire ecosystem is secured via strict backend API proxying, Dockerized for portability, and configured for seamless deployment on **Google Cloud Run** using Artifact Registry. 

I'm incredibly proud of the robust, fault-tolerant design (including high-fidelity API fallbacks) engineered to ensure zero downtime.

Check out the GitHub repo for the complete open-source solution and architecture details!
